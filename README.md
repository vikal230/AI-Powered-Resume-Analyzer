# 🚀 AI Resume Analyzer & Interview Planner

An intelligent web application that helps candidates align their resumes with job descriptions using **Generative AI**. It provides match scores, identifies skill gaps, and generates a custom interview preparation roadmap.

## ✨ Features
1. **AI Analysis:** Uses Google Gemini API to compare resumes with Job Descriptions (JD).
2. **Match Score:** Get a percentage-based compatibility score.
3. **Skill Gap Analysis:** Highlights missing skills (e.g., Tailwind CSS).
4. **Interview Strategy:** Generates custom technical and behavioral questions.
5. **PDF Parsing:** Automatically extracts data from uploaded PDF resumes.
6. **Modern UI:** Clean, dark-themed dashboard built with React and Sass.

## 🛠️ Tech Stack
**Frontend:**
* React 19 (Vite), React Router Dom, Sass, Lucide React

**Backend:**
* Node.js, Express.js, MongoDB, Google Gemini API, Multer, Puppeteer, Zod
